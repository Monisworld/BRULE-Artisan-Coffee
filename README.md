# Brûlé Artisan Coffee – E-Commerce Website

A modern and responsive coffee e-commerce website showcasing artisan coffee products with a clean and premium interface.

## Technologies Used

* HTML (with embedded CSS and JavaScript)

## Features

* Responsive landing page
* Coffee product showcase
* Modern UI design
* Basic interactive elements

## How to Run

1. Download the project file.
2. Open `index.html` in any web browser.
